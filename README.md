# 💰 Controle de Gastos

Projeto desenvolvido em **Python** para registrar e acompanhar gastos pessoais pelo terminal.

---

## 📋 Funcionalidades

- ➕ Adicionar um gasto com nome e valor
- 📄 Ver todos os gastos registrados
- 💵 Ver o total gasto
- 🚪 Sair do programa

---

## 🖥️ Como executar

1. Certifique-se de ter o **Python 3** instalado no seu computador
2. Clone ou baixe este repositório
3. No terminal, execute:

```bash
python controle_de_gastos.py
```

---

## 📸 Exemplo de uso

```
=== CONTROLE DE GASTOS ===
1 - Adicionar gasto
2 - Ver todos os gastos
3 - Ver total gasto
4 - Sair
Escolha uma opção: 1
Nome do gasto (ex: mercado, transporte): Mercado
Valor gasto: R$ 150.00
Gasto adicionado com sucesso!
```
